# SaaS Customer Churn Prediction Analysis

## End-to-End Data Analytics, Machine Learning, SQL & Power BI Project

This project is an end-to-end SaaS customer churn analysis and prediction pipeline built using **Python, Machine Learning, SQL, and Power BI**.

The goal of this project is to help a SaaS business identify customers who are likely to churn, understand the key reasons behind churn, measure revenue risk, and support proactive customer retention decisions.

---

## Project Overview

Customer churn is one of the biggest challenges for SaaS companies because it directly affects recurring revenue, customer lifetime value, and business growth.

Instead of only tracking churn after customers leave, this project focuses on predicting churn risk before it happens. The project combines data cleaning, exploratory data analysis, machine learning, SQL-based business analysis, and Power BI dashboarding to create a complete business intelligence solution.

---

## Business Problem

SaaS companies lose revenue when customers cancel their subscriptions. If the business only reacts after churn happens, it becomes difficult to protect revenue and retain valuable customers.

This project answers key business questions such as:

- Which customers are more likely to churn?
- What customer behaviors are connected with churn?
- Which customer segments create the highest revenue risk?
- How can the business identify high-risk customers earlier?
- What actions can help improve customer retention?

---

## Project Objectives

The main objectives of this project are:

1. Clean and prepare raw SaaS customer data for analysis.
2. Perform exploratory data analysis to understand churn behavior.
3. Create meaningful features for churn prediction.
4. Train and evaluate machine learning models.
5. Select the best-performing model for churn prediction.
6. Export prediction results for business analysis.
7. Use SQL to generate KPIs and business insights.
8. Build a Power BI dashboard for executive-level decision-making.
9. Provide business recommendations to reduce churn and protect revenue.
